# 2026 北京深度游玩全攻略

## 1. 目的地简介
北京是中国的首都，是一座拥有3000多年历史的文化名城。这里是中华文明的重要发祥地，拥有众多世界文化遗产和历史古迹。北京既有庄严的皇家建筑，也有现代化的都市风光，是一座传统与现代交融的城市。适合历史文化爱好者、家庭旅行和第一次来中国的游客。

## 2. 核心景点推荐 (含门票与位置信息)

### 2.1 故宫博物院
* **位置**：北京市东城区景山前街4号
* **门票**：旺季60元/人，淡季40元/人
* **游玩时长**：半天-1天
* **简介**：世界上现存规模最大、保存最为完整的木质结构古建筑群，曾是明清两代的皇家宫殿。建议从午门进入，参观三大殿、后三宫和御花园，感受皇家气派。

### 2.2 天安门广场
* **位置**：北京市东城区天安门广场
* **门票**：免费
* **游玩时长**：1-2小时
* **简介**：世界上最大的城市广场，是中国的象征。广场上有人民英雄纪念碑、毛主席纪念堂等重要建筑。建议清晨观看升国旗仪式，感受庄严氛围。

### 2.3 八达岭长城
* **位置**：北京市延庆区军都山关沟古道北口（距市区约1.5小时车程）
* **门票**：旺季40元/人，淡季35元/人
* **游玩时长**：半天-1天
* **简介**：万里长城的重要组成部分，是明长城中保存最好、最具代表性的一段。建议选择北坡或南坡攀登，欣赏长城蜿蜒曲折的壮丽景色。

### 2.4 颐和园
* **位置**：北京市海淀区新建宫门路19号
* **门票**：旺季30元/人，淡季20元/人（园中园另收费）
* **游玩时长**：半天-1天
* **简介**：中国现存规模最大、保存最完整的皇家园林，被誉为"皇家园林博物馆"。昆明湖、长廊、佛香阁是必看景点，春季桃花盛开时景色最美。

### 2.5 天坛
* **位置**：北京市东城区天坛东里甲1号
* **门票**：旺季15元/人，淡季10元/人（祈年殿等另收费）
* **游玩时长**：2-3小时
* **简介**：明清两代皇帝祭天祈谷的场所，是中国古代祭祀建筑的杰出代表。祈年殿、皇穹宇、圜丘坛是核心景点，建筑气势恢宏。

## 3. 特色餐饮与预算参考

* **北京烤鸭**：必吃美食，推荐全聚德、大董，人均预算 **150-250元**。
* **涮羊肉**：冬季必吃，铜锅涮肉，推荐东来顺、南门涮肉，人均预算 **80-150元**。
* **炸酱面**：北京特色面食，酱香浓郁，人均预算 **20-30元**。
* **豆汁儿配焦圈**：老北京传统早餐，豆汁儿酸中带甜，人均预算 **10-15元**。
* **卤煮火烧**：地道小吃，猪肠、肺头炖煮，人均预算 **20-30元**。
* **糖葫芦**：传统甜品，酸甜可口，人均预算 **5-10元**。

## 4. 住宿区域建议

* **王府井/天安门周边**：市中心，交通便利，靠近景点。酒店预算：**400-800元/晚**。
* **西单/金融街**：商业区，购物方便。酒店预算：**350-700元/晚**。
* **三里屯/国贸**：现代都市氛围，夜生活丰富。酒店预算：**400-800元/晚**。
* **颐和园/中关村**：西北区域，适合参观颐和园的游客。酒店预算：**300-600元/晚**。

## 5. 经典三日行程参考

* **第一天：皇家文化之旅**
  * 上午：参观【天安门广场】，观看升旗仪式。
  * 下午：游览【故宫博物院】，感受皇家宫殿的宏伟。
  * 晚上：在【王府井】购物，品尝北京小吃。
* **第二天：长城与园林**
  * 上午：前往【八达岭长城】，体验万里长城的壮观。
  * 下午：游览【颐和园】，欣赏皇家园林的美景。
  * 晚上：品尝正宗北京烤鸭。
* **第三天：文化与现代**
  * 上午：参观【天坛】，感受古代祭祀文化。
  * 下午：前往【南锣鼓巷】，逛胡同，感受老北京风情。
  * 晚上：在【三里屯】体验北京夜生活。

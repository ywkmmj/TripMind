# 2026 杭州深度游玩全攻略

## 1. 目的地简介
杭州是浙江省省会，古称"临安"，是中国七大古都之一。杭州以西湖闻名于世，被誉为"人间天堂"。这里风景秀丽，人文荟萃，既有江南水乡的婉约之美，又有现代都市的繁华。杭州还是阿里巴巴总部所在地，充满创新活力。适合情侣出游、家庭旅行和文化爱好者。

## 2. 核心景点推荐 (含门票与位置信息)

### 2.1 西湖
* **位置**：杭州市西湖区
* **门票**：免费（部分景点收费）
* **游玩时长**：半天-1天
* **简介**：杭州的标志性景点，由一山（孤山）、两堤（苏堤、白堤）、三岛（小瀛洲、湖心亭、阮公墩）组成。推荐步行或骑行西湖环线，欣赏西湖十景。春季樱花、秋季桂花盛开时最美。

### 2.2 灵隐寺
* **位置**：杭州市西湖区灵隐路法云弄1号
* **门票**：飞来峰45元/人，灵隐寺香花券30元/人
* **游玩时长**：3-4小时
* **简介**：中国著名的佛教寺院，始建于东晋。寺内有大雄宝殿、药师殿等建筑，香火旺盛。飞来峰石刻造像是重要的文物景观。

### 2.3 西溪国家湿地公园
* **位置**：杭州市西湖区天目山路518号
* **门票**：80元/人
* **游玩时长**：半天-1天
* **简介**：中国首个国家湿地公园，以"一曲溪流一曲烟"闻名。可以乘坐电瓶船或摇橹船游览，欣赏湿地生态风光。秋季的芦苇荡景色迷人。

### 2.4 雷峰塔
* **位置**：杭州市西湖区南山路15号
* **门票**：40元/人
* **游玩时长**：1-2小时
* **简介**：因《白蛇传》故事而闻名，是西湖十景之一"雷峰夕照"的所在地。登上塔顶可俯瞰西湖全景。

### 2.5 千岛湖
* **位置**：杭州市淳安县（距市区约2小时车程）
* **门票**：旺季150元/人，淡季120元/人
* **游玩时长**：1天
* **简介**：由1078座岛屿组成的人工湖，湖水清澈见底。可以乘船游览中心湖区或东南湖区，登岛参观。适合喜欢自然风光和水上活动的游客。

## 3. 特色餐饮与预算参考

* **西湖醋鱼**：杭州名菜，酸甜口味，推荐楼外楼、知味观，人均预算 **60-80元**。
* **龙井虾仁**：鲜嫩可口，春季龙井新茶配虾仁，人均预算 **80-120元**。
* **叫化鸡**：传统名菜，泥封烤制，香气四溢，人均预算 **50-70元**。
* **葱包烩**：街头小吃，油条裹葱油炸，配甜面酱，人均预算 **5-8元**。
* **片儿川**：杭州特色面条，雪菜肉丝浇头，人均预算 **15-20元**。
* **西湖藕粉**：传统甜品，细腻爽滑，人均预算 **8-12元**。

## 4. 住宿区域建议

* **西湖周边**：靠近景点，风景优美。酒店预算：**400-800元/晚**。
* **湖滨商圈**：市中心，交通便利，购物方便。酒店预算：**300-600元/晚**。
* **河坊街/吴山广场**：老杭州氛围，美食众多。酒店预算：**250-500元/晚**。
* **西溪湿地周边**：适合休闲度假，环境清幽。酒店预算：**300-600元/晚**。

## 5. 经典三日行程参考

* **第一天：西湖全景游**
  * 上午：漫步【苏堤】，参观【岳王庙】。
  * 下午：乘船游西湖，登【小瀛洲】，参观三潭印月。
  * 晚上：在【湖滨银泰】购物，欣赏西湖音乐喷泉。
* **第二天：文化与自然**
  * 上午：游览【灵隐寺】，感受佛教文化。
  * 下午：前往【西溪国家湿地公园】，体验湿地风光。
  * 晚上：在【河坊街】品尝小吃，逛夜市。
* **第三天：周边深度游**
  * 上午：前往【千岛湖】，乘船游览湖区岛屿。
  * 下午：继续游览千岛湖，品尝湖鲜。
  * 晚上：返回市区，品尝杭州特色美食。
